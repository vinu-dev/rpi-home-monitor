What to do when .local does not work

If your phone or laptop cannot open a .local address, use the IP fallback shown in the product UI.

  1. Open the camera status page and scan or copy the By IP link.
  2. Open the server login page or dashboard and copy the Server address link.
  3. Bookmark the IP URL you used successfully on this WiFi network.
If neither page opens

Check that your device is back on the same home WiFi as the camera and server. If the camera is unavailable, reconnect to the setup hotspot and repeat setup. If the server is unavailable, verify it is powered on and reachable from another device on the same network.
